
การสร้างแอปพลิเคชัน Generative AI ที่มีประสิทธิภาพ จำเป็นต้องเลือก Foundation Model ที่เหมาะสมกับกรณีใช้งานของคุณโดยเฉพาะ ด้วยจำนวนโมเดลที่มีอยู่หลายพันรายการ คุณจึงต้องมีแนวทางที่เป็นระบบในการค้นหา เปรียบเทียบ ปรับใช้ และตรวจสอบว่าโมเดลนั้นตอบโจทย์ความต้องการของคุณ

ลองพิจารณาสถานการณ์ที่คุณกำลังสร้างแชตบอตบริการลูกค้าที่ขับเคลื่อนด้วย AI สำหรับบริษัทค้าปลีก คุณต้องเลือกโมเดลภาษา (language model) ที่สามารถเข้าใจคำถามของลูกค้า ให้คำตอบได้อย่างถูกต้อง และรักษาน้ำเสียงรวมถึงมาตรฐานด้านความปลอดภัยที่เหมาะสม แต่คุณจะเลือกอย่างไรจากแคตตาล็อกโมเดลจำนวนมหาศาลที่มีอยู่ คุณจะรู้ได้อย่างไรว่าโมเดลใดทำงานได้ดีสำหรับความต้องการเฉพาะของคุณ และเมื่อปรับใช้แล้ว คุณจะวัดผลและปรับปรุงประสิทธิภาพอย่างไร

Microsoft Foundry portal มีแพลตฟอร์มที่ครอบคลุมสำหรับเวิร์กโฟลว์ทั้งหมดนี้ คุณสามารถสำรวจโมเดลได้มากกว่า 1,900 รายการจากผู้ให้บริการอย่าง Microsoft, Anthropic, OpenAI, Meta และ Hugging Face คุณสามารถเปรียบเทียบโมเดลโดยใช้เกณฑ์มาตรฐานระดับอุตสาหกรรมในด้านคุณภาพ ความปลอดภัย ต้นทุน และประสิทธิภาพ หลังจากเลือกโมเดลแล้ว คุณจะปรับใช้ไปยัง endpoint เพื่อให้แอปพลิเคชันของคุณเรียกใช้งานได้ สุดท้าย คุณจะประเมินประสิทธิภาพของโมเดลด้วยทั้งเมตริกอัตโนมัติและการทดสอบแบบแมนนวล เพื่อให้มั่นใจว่าโมเดลผ่านข้อกำหนดด้านคุณภาพและความปลอดภัย

ในโมดูลนี้ คุณจะได้เรียนรู้วิธีใช้ Microsoft Foundry portal เพื่อเลือก ปรับใช้ และประเมินโมเดลจากแคตตาล็อกโมเดล คุณจะได้เรียนรู้วิธีตัดสินใจเลือกโมเดลอย่างมีข้อมูลรองรับ ทำความเข้าใจตัวเลือกการปรับใช้ที่แตกต่างกัน และประเมินประสิทธิภาพโมเดลด้วยแนวทางการประเมินหลายรูปแบบ

เมื่อจบโมดูลนี้ คุณจะสามารถ:

- สำรวจและกรองโมเดลในแคตตาล็อกโมเดล
- เปรียบเทียบโมเดลโดยใช้เมตริก benchmark ด้านคุณภาพ ความปลอดภัย ต้นทุน และประสิทธิภาพ
- ปรับใช้โมเดลไปยัง endpoint และทดสอบใน playground
- ประเมินประสิทธิภาพของโมเดลด้วยแนวทางทั้งแบบแมนนวลและแบบอัตโนมัติ
- เข้าใจเมตริกการประเมินแบบต่างๆ และรู้ว่าควรใช้เมื่อใด

หมายเหตุ

เราเข้าใจว่าผู้เรียนแต่ละคนมีรูปแบบการเรียนรู้ที่ต่างกัน คุณสามารถเลือกเรียนโมดูลนี้ในรูปแบบวิดีโอ หรืออ่านเนื้อหาในรูปแบบข้อความและภาพประกอบก็ได้ โดยเนื้อหาแบบข้อความจะมีรายละเอียดมากกว่าวิดีโอ ดังนั้นในบางกรณี คุณอาจใช้เนื้อหาข้อความเป็นสื่อเสริมจากการรับชมวิดีโอ
