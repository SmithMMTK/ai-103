
Developers ที่สร้าง AI solutions ด้วย Microsoft Foundry จำเป็นต้องทำงานร่วมกับทั้ง services และ software frameworks หลายแบบ เมื่อพัฒนา generative AI chat application คุณสามารถเลือก options ที่หลากหลายเพื่อเขียน code ที่จำเป็นต่อการสร้าง AI apps ที่มีประสิทธิภาพบน Azure

ในโมดูลนี้ คุณจะได้เรียนรู้วิธีเลือก endpoint, SDK, authentication และ chat API options ที่เหมาะสม เพื่อสร้าง generative AI chat application

> [!NOTE]
> เราเข้าใจว่าผู้เรียนแต่ละคนมีรูปแบบการเรียนรู้ที่ต่างกัน คุณสามารถเลือกเรียนโมดูลนี้ในรูปแบบ video-based หรืออ่านเนื้อหาในรูปแบบ text และ images ก็ได้ โดยเนื้อหาแบบข้อความมีรายละเอียดมากกว่าวิดีโอ ดังนั้นในบางกรณีคุณอาจต้องการใช้อ้างอิงเป็น supplemental material ควบคู่กับการรับชมวิดีโอ
