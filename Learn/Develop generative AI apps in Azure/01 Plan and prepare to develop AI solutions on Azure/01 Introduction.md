
การเติบโตของการใช้งาน **ปัญญาประดิษฐ์ (AI)** โดยทั่วไป และ **Generative AI** โดยเฉพาะ ทำให้นักพัฒนาจำเป็นต้องสร้างโซลูชัน AI ที่มีความครบถ้วนและครอบคลุมมากขึ้น โดยโซลูชันเหล่านี้มักต้องผสานการทำงานระหว่าง

- โมเดล Machine Learning
- บริการ AI สำเร็จรูป
- เทคนิค Prompt Engineering
- โค้ดที่พัฒนาขึ้นเอง (Custom Code)

Microsoft Azure มีบริการหลากหลายที่สามารถนำมาใช้ในการสร้างโซลูชัน AI ได้ อย่างไรก็ตาม ก่อนเริ่มโครงการพัฒนาแอปพลิเคชัน AI ควรทำความเข้าใจตัวเลือกต่าง ๆ ที่มีอยู่ ทั้งในด้านบริการ เครื่องมือ และเฟรมเวิร์ก รวมถึงหลักการและแนวปฏิบัติที่ช่วยเพิ่มโอกาสความสำเร็จของโครงการ

โมดูลนี้จะอธิบายประเด็นสำคัญที่ควรพิจารณาในการวางแผนโครงการพัฒนา AI และแนะนำ **Microsoft Foundry** ซึ่งเป็นแพลตฟอร์มแบบครบวงจรสำหรับการพัฒนา AI บน Microsoft Azure.
