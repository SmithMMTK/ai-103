ในแบบฝึกหัดนี้ คุณจะ build และ deploy AI agent ด้วย Microsoft Foundry Agent Service โดยจะสร้าง agent ตั้งค่า instructions และ tools เชื่อมต่อกับ Visual Studio Code extension และนำ agent ไปใช้ในแอปพลิเคชัน

> [!NOTE]
> หากคุณยังไม่มี Azure subscription และต้องการลองใช้งาน Microsoft Foundry คุณสามารถ [สมัครบัญชี](https://azure.microsoft.com/pricing/purchase-options/azure-account?cid=msft_learn) ซึ่งมีเครดิตให้ในช่วง 30 วันแรก
